# مشروع Summarize - منصة الملفات الدراسية

هذا هو الكود المصدري لمنصة Summarize. تم تصميم الواجهة لتكون سلسة وسريعة باستخدام تقنيات حديثة.

## كيفية التشغيل
بما أنك لا تحتاج إلى تثبيت أي برامج معقدة، يمكنك تشغيل الموقع فوراً:
1. افتح المجلد الذي يحتوي على الملفات.
2. انقر نقراً مزدوجاً على ملف `index.html` لفتحه في المتصفح.

## ملاحظة مهمة
هذا التطبيق يستخدم **localStorage** بشكل حصري لتخزين البيانات محلياً على جهازك. جميع الملفات والمستخدمين والإعدادات تُحفظ في متصفح الويب الخاص بك فقط.

## التقنيات المستخدمة
- **HTML5 & CSS3**
- **Tailwind CSS** (للتصميم العصري)
- **JavaScript** (للتعامل مع البيانات)
- **Lucide Icons** (للأيقونات)
